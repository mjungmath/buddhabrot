# buddhabrot

Playing around with [buddhabrot](https://en.wikipedia.org/wiki/Buddhabrot), a close relative of the Mandebrot set.

This repo contains two versions to speed things up. Intermediate results will be stored in a file so that the computation can be continued anytime.
